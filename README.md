# Dependencies

- Tailwindcss
  npm install -D tailwindcss postcss autoprefixer
  npx tailwindcss init -p

- React Router
  npm i react-router-dom

- React icons
  npm i react-icons --save

- TipTap
  npm install @tiptap/react @tiptap/pm @tiptap/starter-kit
